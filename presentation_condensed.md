# Сэмжид Хужирт Wellness Resort Вэбсайт Танилцуулга

## Оршил ба Урам Зориг

### Төсөл
- Бүрэн боломжтой хоёр хэл дээрх амралтын газрын захиалгын вэбсайт
- Next.js 14, TypeScript, Supabase, Tailwind CSS ашигласан

### Асуудал ба Шийдэл
- **Асуудал**: Гар аргаар захиалах процесс хүнд сурталтай, олон алхамтай
- **Шийдэл**: Автомат онлайн захиалга, бодит цагийн боломж, 24/7 үйлчилгээ

---

## Төслийн Тойм ба Онцлогууд

### Үндсэн Онцлогууд
- **Онлайн Захиалга**: 3 алхамтай процесс, QPay интеграци
- **Админ Самбар**: Аналитик, төлбөрийн удирдах, чат систем
- **Хоёр Хэл Дэмжлэг**: Монгол ба Англи
- **Бодит Цагийн Боломж**: Давхар захиалгыг зогсоох

### Технологи
- **Frontend**: Next.js 14, React, TypeScript
- **Backend**: Supabase, Resend API
- **Deployment**: Vercel
